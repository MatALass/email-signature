# Privacy & Anti-Spam Considerations

This repository is public.

Publishing personal contact information (phone number, email address,
LinkedIn URL) may increase exposure to:

-   Spam
-   Automated scraping bots
-   Targeted phishing attempts
-   Data harvesting

------------------------------------------------------------------------

## Recommended Safer Practices

### 1. Use an Alias Email

Create a dedicated public-facing email address for: - Portfolios -
GitHub repositories - Email signatures

### 2. Reduce Phone Exposure

Consider: - Removing the phone number - Displaying phone only in direct
email replies - Using a business-only number

### 3. Link to a Contact Page

Instead of publishing your direct email, link to: - A portfolio contact
form - A business landing page - A filtered contact gateway

------------------------------------------------------------------------

## Icon Hosting & Tracking

Email clients may load remote images automatically. Each image request
can act as a tracking signal.

To minimize tracking:

-   Use minimal external assets
-   Avoid large hosted images
-   Accept that some clients proxy images (e.g., Gmail)

------------------------------------------------------------------------

## Data Responsibility

If adapting this template for a company:

-   Ensure compliance with GDPR or local privacy laws
-   Avoid exposing internal-only contact data
-   Verify branding and data usage policies

------------------------------------------------------------------------

Privacy is part of engineering quality. Design intentionally.
